# AniEngine

AniEngine is a lightweight C++ runtime focused on real-time animation systems.

See [AniEngineDemo](https://github.com/WangSimiao2000/AniEngineDemo) for runnable examples.

Planned features:

- Scene Graph
- Transform Hierarchy
- Timeline
- Animation Clip
- Keyframe Interpolation
- Curve Evaluation
- Blend Tree
- Animation Layers
- State Machine
- Constraints
- Skeletal Animation
